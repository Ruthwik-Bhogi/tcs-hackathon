# Hospital AI Digital Twin

## Generate realistic data

```bash
python data/synthetic_generator.py
```

## Train model

```bash
python ml/train.py
```

## Run dashboard

```bash
streamlit run app.py
```

---

## Features

- Digital Twin Simulation
- ICU Occupancy Monitoring
- ER Load Prediction
- Surge Detection
- Synthetic Hospital Data
- XGBoost Forecasting
- Streamlit Dashboard
- Optional ROCm + vLLM Narrator
